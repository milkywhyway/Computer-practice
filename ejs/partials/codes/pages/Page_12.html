Пояснення

Для роботи з JavaScript-проектами існує таск-менеджер npm. Його можно завантажити за посиланням https://nodejs.org/en/download/ . Для створення проекту потрібно ініціалізувати npm у папці за допомогою консольної команди npm init. Npm може виконувати таку ж функцію, як і Makefile, його скрипти будуть записани в файлі package.json у об'єкті з ключем scripts. Для запуску скриптів треба ввести в консоль ключаві слова npm run та назву скрипта. Наприклад, з набором скриптів у данному проекті, можна запустити проект командою npm run start. Окрім цього npm дозволяє керувати данними проекту та додавати/видаляти сторонні бібліотеки або фреймворки.

Про всяк випадок в кореневій папці проекту лежить Makefile, що повторює скрипти з файлу package.json.
